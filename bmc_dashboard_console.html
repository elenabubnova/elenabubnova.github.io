BMC ITSM Dashboards
Improving web experience

My Role: Lead Product Designer

Background

User research indicated that the current ITSM dashboard lacks functionality and does not cater to the diverse needs of different personas, presenting a confusing mix of information. The objective was to redesign the dashboard into an actionable tool tailored for agents, emphasizing their day-to-day tasks and prioritizing key activities.

Tasks

  1. Enhance actionability of the dashboard

  2. Provide a quick overview of daily updates

  3. Implement persona-based configurations

  4. Establish a new information hierarchy

  5. Conduct thorough testing

Persona

Solution

Asset Console

Background

BMC Helix ITSM's Asset Console is a pivotal feature that facilitates comprehensive IT asset management. However, user feedback indicates that UI is outdated and requires modernization. Some enhancements could significantly improve user experience and functionality.

Tasks

  1. Research

  2. Make actionable dashboard

  3. Simplify bundle processes

  4. Update UI

  5. Find major drawnbacks

Before

Interview

Console Manager who is working with it day-to-day, we identified many areas for improvements but with infinity map focused on major one. We also did Competitive Analysis and examined similar platforms to identify innovative features.

  1. Performance Issues:

    Reports of sluggishness, especially during navigation between tabs, have been noted.

  2. Search and Filter Limitations:

    The Asset Console's search and filter functionalities are perceived as less efficient compared to other tools, leading to reduced usage.

  3. Many repetitive manual actions,

Improvements

Based on research and requirements we found that users appreciate the intuitive navigation and alignment with service management processes along with integration capabilities. While keeping that we improved:

  1. Real-Time Data Visualization: 

    Implemented dynamic charts and graphs.

  2. Customizable Widgets:

    Tailored dashboard to display relevant metrics and information.

  3. Automated Workflows:

    Introduced automation to reduce manual interventions.

  4. Assets Grouped in Tab Menu:

    Organized assets by categories for better navigation.

  5. 3D Map: 

    Added a 3D map to easily locate assets.

  6. Classic Table View: 

    Kept the classic table view for smooth user adoption.
