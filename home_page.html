BMC Home Page

My Role: UX / Visual Design / Web

Objective

The focus of this redesign is to use the home page body to tell an impactful story about where IT management is headed and how BMC can help you get there. To support this, we had 2 priority objectives for the design.

  1. Generate awareness for the Autonomous Digital Enterprise (ADE) and the role BMC plays in helping to achieve it.

  2. Portray BMC as a relevant and important provider of SAAS IT management products and solutions.

Requirements

  1. Keep it short. Intentionally short! Less was the goal here.

  2. Provide a mechanism to display time-sensitive announcements.

  3. Create a design framework that can accommodate targeted content (personalization) and translated content.

Success measures
  1. Qualitative

    User testing: Visitors understand the concept of “Autonomous Digital Enterprise”
    Visitors understand that BMC provides SAAS solutions
    Visitors validate the page feels modern and the content is relevant

  2. Qualitative

    Maintain or lower current home page bounce rate (~51%)

Solution

The web platform has a lot of limitations, including work with responsive design, for example, global navigation not expected to be changed.

The part of the concept was creating a hierarchy with main messages. It was solved successfully with the different sizes tile system, which includes a carousel and simple CSS animation. Each section leading you on 4 pages accordingly.

The web-site got a lot of recognition internally and externally.
