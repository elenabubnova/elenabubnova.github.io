BMC Helix Portal

The BMC Helix Portal, a critical interface for launching and managing applications, required an update to improve usability and efficiency while aligning with the new BMC branding.

Challenges

  1. Visual clutter in product cards
  2. Insufficient filtering options
  3. Absence of personalization features
  4. Scalability issues impacting user experience

Improvements

  1. Created new product Identity
  2. Implemented micro-interactions to reduce visual clutter
  3. Integrated a global search feature into the navigation, as opposed to a local search
  4. Added category-based filtering options
  5. Developed a mobile version for enhanced accessibility

Addressed challenges with the favorites button due to limitations in data storage.
