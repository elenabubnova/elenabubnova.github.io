About Product page

BMC Helix applications previously lacked a dedicated About Product page. The current version was only accessible through a dropdown menu, and new features were announced in a pop-up window upon updating.

New Design Features

  1. Upselling banner with the latest version
  2. Structured content for improved readability
  3. Interactive elements (carousel)
  4. Crosslinks to relevant documentation
  5. Preview of upcoming updates
  6. Updated branding

The new design has been templated and integrated into the Design System.
