Integration Helix GPT into the Web Platforms

Part 1

We examined the coexistence of two primary touchpoints for initiating searches, recognising that search is a critical user goal for pages like Documentation: within the global navigation bar and via a prominent search bar on the webpage banner.

Improvements

  • Added a prominent Hero bar search bar on the landing page, emphasising search as a primary user objective.

Global search button in the navigation bar

Helix GPT as a chat

Some products lack flexibility for significant changes, requiring us to implement workarounds where HelixGPT operates as a pop-up chat view. This design will not interfere with or interact with elements on the user's screen at that moment, although it will have visibility into the user's current context.
